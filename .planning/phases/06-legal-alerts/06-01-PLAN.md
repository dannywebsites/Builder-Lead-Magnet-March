<objective>
Build conditional legal and tax alerts that fire based on calculator inputs/outputs and display on the results screen. Four alerts: Gross Draw Warning (always), Irish Two-Thirds VAT Rule (conditional), Efficiency Cap (when staff > 0), and CIS/RCT Subcontractor Warning (always). Alert logic lives in a pure function module; alert copy in centralized constants; UI uses amber-styled cards inserted between the Financial Anchors and Sales Pipeline sections.

Purpose: Tradespeople need to know about legal and tax gotchas that could bite them -- personal tax not covered, VAT rate issues, efficiency assumptions, and subcontractor withholding tax. These alerts protect the user from making decisions based on incomplete information.

Output: Working alert system with 4 alerts, unit tests, and visual integration on the results screen.

    **Step 2: Create `src/lib/calculator/alerts.ts`** (per D-07, D-08)

    ```typescript
    import type { CalculatorInput, CalculatorOutput } from "./types";
    import { ALERT_COPY } from "../results/alert-copy";

    export type AlertKey = "gross-draw" | "two-thirds-rule" | "efficiency-cap" | "cis-rct";

    export interface Alert {
      key: AlertKey;
      title: string;
      body: string;
    }

    export function getTriggeredAlerts(
      input: CalculatorInput,
      _output: CalculatorOutput,
    ): Alert[] {
      const alerts: Alert[] = [];

      // ALRT-01: Gross Draw Warning — always fires for both entity types (per D-09)
      alerts.push({
        key: "gross-draw",
        title: ALERT_COPY["gross-draw"].title,
        body: ALERT_COPY["gross-draw"].body,
      });

      // ALRT-02: Irish Two-Thirds Rule — vatRate === 0.135 AND directCostPct > 0.66 (per D-10)
      if (input.vatRate === 0.135 && input.directCostPct > 0.66) {
        alerts.push({
          key: "two-thirds-rule",
          title: ALERT_COPY["two-thirds-rule"].title,
          body: ALERT_COPY["two-thirds-rule"].body,
        });
      }

      // ALRT-03: Efficiency Cap — when staffCount > 0 (per D-11)
      if (input.staffCount > 0) {
        alerts.push({
          key: "efficiency-cap",
          title: ALERT_COPY["efficiency-cap"].title,
          body: ALERT_COPY["efficiency-cap"].body,
        });
      }

      // ALRT-04: CIS/RCT Warning — always fires (per D-12)
      alerts.push({
        key: "cis-rct",
        title: ALERT_COPY["cis-rct"].title,
        body: ALERT_COPY["cis-rct"].body,
      });

      return alerts;
    }
    ```

    Note: `_output` parameter is kept for future extensibility but unused currently.
